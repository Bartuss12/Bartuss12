What is it?

Clogger lets you scan and save 433Mhz signals,

store them and send them to open doors, gates, etc.



Note

This project should not be used in any illegal way.

It is created for pure educative purpouse.



Part list

-arduino (or atmel chip with arduino bootloader)

-protoboard (to solder components onto)

-433mhz transmitter and reciver

-2 microswitches

-cables (unless you make connections on board)

-some source of voltage (I have chosen lipo battery from an old phone)



Updates

Version 1

2 buttons. No.1 makes the device read incoming signals and store them in array.

No.2 makes the device send all the signals stored in the array.
